# 地图与开发计划 v0.1

状态：待确认  
范围：5 张地图、暗雷表、地图元素、开发阶段

## 1. 地图技术规格建议

Demo 建议统一采用 Tile 地图：

- Tile 尺寸：32x32 像素。
- 玩家视野：约 20x15 Tile。
- 地图数据：二维数组或 JSON。
- 地形层：地面层、装饰层、碰撞层、事件层。
- 遇敌区域：按区域配置暗雷概率与宠物池。

推荐移动方式：

- 桌面：方向键或 WASD。
- 移动端：虚拟方向键。
- 每移动 1 格进行一次暗雷判定。
- 安全区、道路、营地不触发暗雷。

暗雷概率建议：

| 区域类型 | 每步遇敌率 |
|---|---:|
| 安全道路 | 0% |
| 普通草丛/浅滩/沙地 | 5% |
| 深处危险区 | 8% |
| 稀有巢穴 | 12% |

## 7. 地图进度目标

每张地图都有 3 类目标：

| 目标 | 说明 |
|---|---|
| 探索目标 | 到达地图关键点，例如祭坛、洞穴、破船 |
| 收集目标 | 捕获该地图至少 3 只不同宠物 |
| 战斗目标 | 击败区域首领 |

完成全部地图目标后，Demo 显示结算：

- 已发现宠物数量。
- 已拥有宠物数量。
- 合成次数。
- 捕获成功率。
- 击败首领数量。

## 8. 暗雷战斗编组

普通遭遇：

- 60%：1 只野生宠物。
- 30%：2 只野生宠物。
- 10%：3 只野生宠物。

稀有巢穴：

- 50%：1 只高出现率宠物。
- 35%：2 只同属性宠物。
- 15%：1 只进化体稀有宠物。

敌方个体 Lv：

- 每张地图以营地为安全中心。
- 离营地越近，遇到的敌方宠物个体 Lv 越低。
- 离营地越远，遇到的敌方宠物个体 Lv 越高。
- Demo 采用曼哈顿距离换算：营地附近约 Lv1，地图深处约 Lv9-Lv10。

首领战：

- 当前地图属性完全体 Lv10 三只。
- 首领持有的完全体 Lv10 宠物应用首领属性修正，当前 Demo 倍率为 1.4 倍。
- 三只首领宠物均不可捕获。

## 9. 地图事件

Demo 建议先做轻量事件，不做复杂任务链：

| 事件 | 作用 |
|---|---|
| 治疗点 | 恢复主角和队伍 |
| 一键回城 | 非战斗状态下立刻传送回当前地图营地并恢复队伍 |
| 仓库点 | 管理宠物 |
| 合成师 | 打开合成界面 |
| 采集点 | 获得捕获石、治疗果、属性诱饵 |
| 宝箱 | 一次性奖励 |
| 祭坛 | 触发首领战 |
| 木牌 | 显示简短提示 |

## 10. 保存系统

Web Demo 建议使用浏览器本地存储：

- 保存队伍。
- 保存仓库。
- 保存图鉴发现状态。
- 保存地图解锁状态。
- 保存首领击败状态。
- 保存背包道具。

需要提供“重新开始”按钮，并二次确认。

## 11. 开发技术建议

如果你同意进入开发，我建议采用以下技术路线：

| 模块 | 建议 |
|---|---|
| 前端框架 | React + Vite |
| 语言 | TypeScript |
| 样式 | CSS Modules 或普通 CSS |
| 地图渲染 | HTML/CSS Grid 或 Canvas，Demo 先用 Grid 更快 |
| 战斗逻辑 | 独立 TypeScript 数据与函数 |
| 数据 | JSON/TS 常量表 |
| 保存 | localStorage |

如果目标是最快看到可玩版本，也可以使用原生 HTML/CSS/JavaScript 单文件。但考虑宠物、地图、战斗、仓库、合成系统较多，React + TypeScript 会更适合维护。

## 12. 开发里程碑

### 里程碑 1：最小可玩版

目标：玩家能移动、遇敌、战斗、捕获。

- 项目初始化。
- 主地图渲染。
- 主角移动与碰撞。
- 暗雷判定。

- 宠物普通技能。
- 捕获成功/失败。
- 战斗胜利与失败。

### 里程碑 2：收集与养成

目标：宠物收集闭环成立。

- 队伍管理。

- 宠物经验成长。
- 捕获道具与恢复道具。

### 里程碑 3：合成与五地图

### 里程碑 4：体验打磨

目标：从功能 Demo 变成完整可试玩 Demo。

内容：

- 战斗动画与日志。

- 移动端适配。

## 13. 开发验收标准

第一版开发完成时，至少应满足：

- 打开网页可直接开始游戏。
- 玩家可以在地图移动。
- 暗雷可以正常触发。
- 战斗可以完整结算。
- 玩家最多携带 3 只宠物战斗。
- 可以捕获宠物。
- 可以查看队伍和仓库。
- 可以合成宠物。
- 完全体不能继续合成。
- 五张地图可以解锁和探索。
- 每张地图有对应属性生态。
- 关闭网页后重新打开能继续进度。
