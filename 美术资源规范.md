# 美术资源规范

## 宠物模型

- 新版宠物模型源文件使用 `512×512 RGBA PNG`，必须包含真实透明背景。
- 宠物主体保持在画布中央，脚底尽量靠近统一基线，四周需要保留透明边距。
- 图片只包含宠物主体，不包含场景、地面、文字或不透明背景。
- 游戏根据使用位置将源图等比例缩放至 `64×64` 或 `128×128` 显示，并使用 `object-fit: contain` 防止变形。
- 新版宠物图片使用浏览器高质量缩放；现有程序化宠物、地图瓦片、人物和 UI 图标继续沿用原生 `64×64` 最近邻像素渲染。
- 图片替换只绑定宠物内部 ID，不修改战斗、捕获、升级、合成或存档逻辑。

## 其他资源

- 地图瓦片、人物、UI 图标和现有程序化像素资源继续以原生 `64×64` 为基础尺寸。
- 大型地图对象继续使用 64 的整数倍尺寸。
